# GT2 Codes Translator
This HTML page allows to convert GameShark codes among different versions of PlayStation game Gran Turismo 2 (GT Mode/Simulation disk). The code is a refactored/reissued version of code originally found in [this](https://web.archive.org/web/20021201185852/http://www.geocities.com/gt2toxs/gt/translator/gt2codetr.html) legacy web page.
